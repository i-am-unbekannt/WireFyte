### Installation:
* Linux:
  * `sudo apt update -y`
  * `sudo apt install python3 python3-pip git -y`
  * `pip3 install colorama scapy`
  * `git clone https://github.com/i-am-unbekannt/WireFyte`
  * `cd WireFyte/`
  * `sudo bash WireFyte.sh`

* Termux:
  * `pkg update -y`
  * `pkg install python3 python3-pip git -y`
  * `pip3 install colorama scapy`
  * `git clone https://github.com/i-am-unbekannt/WireFyte`
  * `cd WireFyte/`
  * `bash WireFyte.sh`
